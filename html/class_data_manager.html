◆ DataManagerMT

friend class DataManagerMT
friend

Class used to use the DataManager class in multithreading mode.

A Singleton object is not thread safe.
To be thread safe the strategy usually adopted by geant is to use a different file for each working thread.

With a singleton we cannot use multiple files so we define a friend class DataManagerMT (another singleton) that is able to create multiple instances of DataManager, one for each working thread
