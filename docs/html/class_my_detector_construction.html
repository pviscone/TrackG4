Compute the positions of the center of the modules on a given axis.

Parameters
nOfModulesOnAxisNumber of modules on the axis
moduleSizeSize of the module on a given axis

Returns
std::vector<G4double> Vector that cointains the position of the modules on a given axis

If the number of units is odd then it add to the position vector 0 and \( \pm \) i*(size+spacing)/2 where i is the loop index and start form 1 to nOfUnits+1/2
If the number of units is even then it add to the vector \( \pm \) (i+1/2)*(size+spacing)/2 where i is the loop index and start form 0 to nOfUnits/2
